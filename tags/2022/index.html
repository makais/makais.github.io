LayoutGAN Prototype

A model trained on residential floor plans was used to create a sketch-based application for automatically drawing floor plans from building outlines. Program indicated by color, and the user could specify locations for doors and windows. The sketch app also had simple orthogonal and snap functionality. The model created plans in real-time while the user was drawing to give immediate feedback. It was then fed into models used to identify structural elements for downstream use in a building systems design prototype that used the Concetto platform.
